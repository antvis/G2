使用自定义函数的方式来进行数据的加工和处理，这是最为灵活的一种方式。

## 开始使用

```ts
chart.data({
  type: 'custom',
  callback: (data) => {
    /* your code */
    return [];
  },
});
```

## 选项

| 属性     | 描述                                 | 类型                 | 默认值     |
| -------- | ------------------------------------ | -------------------- | ---------- |
| callback | 函数，传入当前数据，输出处理后的数据 | `(data: any) => any` | `(d) => d` |
