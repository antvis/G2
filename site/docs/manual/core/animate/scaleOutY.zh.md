`scaleOutY` 单个图形沿着 y 方向的消失动画。影响 `transform.scale` 属性。

## 开始使用

<img alt="scaleOutY" src="https://gw.alipayobjects.com/mdn/rms_f5c722/afts/img/A*L6mkQa3aG64AAAAAAAAAAABkARQnAQ" width="400" />

```ts
chart
  .interval()
  /* ... */
  .animate('exit', { type: 'scaleOutY' });
```
